# Pixel Wears

This is an e commerce project, Built with React Js.
Animation is done with Framer motion and Global State Management is handled with framer motion

[Pixel Wears](https://pixel-wears.vercel.app/)

## Tech Used
* React Js V18
* React Router v6
* Styled Components
* Redux Toolkit
* React Icons Package
